Git e github

Definições

    
    Git -> version control system (free e open source)
    Github -> website to host repositories
    Forma de trackear as mudanças no código
    Diretório -> pasta do computador
    Terminal ou command line -> interface para comandos em texto
    CLI -> command line interface (cmd)
    cd -> change directory
    repository -> Project or folder where the project is kept

Instalação

    
    Recomenda-se criar o README.md direto no github
    Baixar pelo [Link para donwload] e não usar o powershell para instalar, usar o git bash

Comandos básicos

    
    clone -> baixar do github para a máquina
    add -> track changes in Git
    commit -> save files in Git
    push -> commits to a remote repo
    pull -> download changes from remote repo

Básico

    
    
  1. git status -> mostra todos os arquivos atualizados, criados ou excluídos que não foram salvos em um commit [untracked files significa que o arquivo git ainda não sabe dele, então precisa avisar o git para trackear com git add]
        
    • ou por ex: git add index.html
    
        
  2. git add . [vai adicionar todos os arquivos, tanto os untracked quanto os modified]
    3. 
    
  3. Para usar algum diretório já existente: git clone git@github.com:user/reponame.git [SSH key]
    4. 
    
  4. git commit -m "Algum título do commit" -m "alguma descrição do commit" 
[o primeiro -m é a mensagem (obrigatória), e o segundo é a descrição (opcional)]
[ainda está apenas na máquina local, o commit não foi "subido" ainda]
    5. 
    
  5. git push -u origin master: para enviar ao github
      
        
    • [-u] serve para definir como padrão, depois não precisa mais
      • 
        
    • [origin] -> opção que dá a localização do repositório
      • 
        
    • [master] -> branch que vai receber o push

Se o projeto tiver sido criado localmente


    
  • git remote add origin git@github.com:user/reponame.git para se conectar
    • 
        
  • git remote -v: para verificar a qual está conectado
    • 
        
  • git remote remove origin: para se desconectar
